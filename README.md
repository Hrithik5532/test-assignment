# Banking Call Center Analysis System - Setup Guide

## Overview

This POC system analyzes banking customer service calls using AI/ML models for:
- **Speech-to-Text**: Convert audio to transcript
- **Intent Classification**: Identify customer purpose
- **Requirement Detection**: Auto-create tickets
- **Sentiment Analysis**: Understand customer emotions
- **Agent Scoring**: Rate agent performance
- **Database Storage**: Track all interactions

---

## Prerequisites

- Python 3.8 or higher
- pip package manager
- 4GB+ RAM (8GB+ recommended)
- GPU optional (speeds up processing)
